CSS How To

This section contains practical examples and tutorials on how to implement various CSS techniques and effects.

Common CSS How-To Examples:

How to Center Elements

Learn different ways to center elements horizontally and vertically.

/* Center horizontally */
.center {
    margin: 0 auto;
    width: 200px;
}

/* Center vertically and horizontally */
.center-all {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}

How to Create Responsive Design

Make your website work on all devices using media queries.

/* Mobile first approach */
.container {
    width: 100%;
    padding: 10px;
}

/* Tablet */
@media (min-width: 768px) {
    .container {
        width: 750px;
        margin: 0 auto;
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .container {
        width: 1200px;
    }
}

How to Create Animations

Add smooth animations and transitions to your elements.

/* Hover animation */
.button {
    background: #4CAF50;
    transition: background 0.3s ease;
}

.button:hover {
    background: #45a049;
}

/* Keyframe animation */
@keyframes slideIn {
    from { transform: translateX(-100%); }
    to { transform: translateX(0); }
}

.slide {
    animation: slideIn 0.5s ease-out;
}

How to Style Forms

Create beautiful and functional form elements.

/* Input styling */
input[type="text"] {
    width: 100%;
    padding: 12px 20px;
    margin: 8px 0;
    border: 2px solid #ddd;
    border-radius: 4px;
    box-sizing: border-box;
}

input[type="text"]:focus {
    border-color: #4CAF50;
    outline: none;
}
