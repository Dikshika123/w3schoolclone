CSS RGB Colors

An RGB color value represents colors using red, green, and blue components.

In CSS, RGB values are specified using the format:

rgb(red, green, blue)

Each parameter (red, green, blue) defines the intensity of the color with a value between 0 and 255.

RGB Color Examples:

rgb(255, 0, 0)

rgb(0, 0, 255)

rgb(0, 255, 0)

rgb(128, 0, 128)

How RGB Works:

  • Red: rgb(255, 0, 0) - Maximum red, no green, no blue
  • Green: rgb(0, 255, 0) - No red, maximum green, no blue
  • Blue: rgb(0, 0, 255) - No red, no green, maximum blue
  • Black: rgb(0, 0, 0) - No red, no green, no blue
  • White: rgb(255, 255, 255) - Maximum red, green, and blue
