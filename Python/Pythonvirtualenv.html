Python VirtualEnv

Virtual environments allow you to create isolated Python environments for different projects.

What is a Virtual Environment?

A virtual environment is an isolated Python environment that allows you to install packages without affecting the system Python installation.

Create Virtual Environment

# Create virtual environment
python -m venv myenv

# Or specify Python version
python3 -m venv myenv

Activate Virtual Environment

# Windows
myenv\Scripts\activate

# Linux/Mac
source myenv/bin/activate

Deactivate Virtual Environment

# Deactivate (works on all platforms)
deactivate

Install Packages in Virtual Environment

# Activate virtual environment first
# Then install packages
pip install requests
pip install numpy
pip install pandas

Create Requirements File

# Generate requirements.txt
pip freeze > requirements.txt

# Install from requirements.txt
pip install -r requirements.txt

Virtual Environment with Different Python Version

# Create virtual environment with specific Python version
python3.9 -m venv myenv39
python3.10 -m venv myenv310

Virtual Environment Best Practices

  • Always use virtual environments for projects
  • Don't commit virtual environment to version control
  • Use descriptive names for virtual environments
  • Keep requirements.txt up to date
  • Use different virtual environments for different projects

Virtual Environment Commands

# List virtual environments
conda env list

# Create virtual environment with conda
conda create -n myenv python=3.9

# Activate conda environment
conda activate myenv

# Deactivate conda environment
conda deactivate

Virtual Environment Structure

myenv/
├── Scripts/          # Windows activation scripts
├── bin/              # Linux/Mac activation scripts
├── include/          # Header files
├── lib/              # Python packages
└── pyvenv.cfg        # Configuration file

Virtual Environment with Poetry

# Install Poetry
pip install poetry

# Create new project
poetry new myproject

# Install dependencies
poetry install

# Add new dependency
poetry add requests

# Activate virtual environment
poetry shell

Virtual Environment with Pipenv

# Install Pipenv
pip install pipenv

# Create Pipfile
pipenv install

# Install packages
pipenv install requests

# Activate virtual environment
pipenv shell
