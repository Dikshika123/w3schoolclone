Python Join Lists

Join Two Lists

There are several ways to join, or concatenate, two or more lists in Python.

One of the easiest ways are by using the + operator.

list1 = ["a", "b", "c"]
list2 = [1, 2, 3]

list3 = list1 + list2
print(list3)

Another way to join two lists are by appending all the items from list2 into list1, one by one:

list1 = ["a", "b", "c"]
list2 = [1, 2, 3]

for x in list2:
    list1.append(x)

print(list1)

Or you can use the extend() method, which purpose is to add elements from one list to another list:

list1 = ["a", "b", "c"]
list2 = [1, 2, 3]

list1.extend(list2)
print(list1)

Join Multiple Lists

You can join multiple lists using the + operator:

list1 = ["a", "b"]
list2 = [1, 2]
list3 = ["x", "y"]

combined = list1 + list2 + list3
print(combined)

Join Lists Using extend()

You can join multiple lists using extend():

list1 = ["a", "b"]
list2 = [1, 2]
list3 = ["x", "y"]

list1.extend(list2)
list1.extend(list3)
print(list1)

Join Lists Using List Comprehension

You can use list comprehension to join lists:

list1 = ["a", "b"]
list2 = [1, 2]

combined = [item for sublist in [list1, list2] for item in sublist]
print(combined)

Join Lists Using itertools.chain()

You can use itertools.chain() for efficient joining:

import itertools

list1 = ["a", "b"]
list2 = [1, 2]
list3 = ["x", "y"]

combined = list(itertools.chain(list1, list2, list3))
print(combined)

Join Lists with Conditions

You can join lists with specific conditions:

list1 = [1, 2, 3, 4, 5]
list2 = [6, 7, 8, 9, 10]

# Join only even numbers
even_numbers = [x for x in list1 + list2 if x % 2 == 0]
print(even_numbers)

Join Lists and Remove Duplicates

You can join lists and remove duplicates:

list1 = ["a", "b", "c"]
list2 = ["b", "c", "d"]

# Join and remove duplicates
unique_items = list(set(list1 + list2))
print(unique_items)

Join Lists with Separator

You can join lists with a separator (for strings):

list1 = ["apple", "banana"]
list2 = ["cherry", "orange"]

# Join with comma separator
joined_string = ", ".join(list1 + list2)
print(joined_string)

Join Lists Using Unpacking

You can use unpacking to join lists:

list1 = ["a", "b"]
list2 = [1, 2]
list3 = ["x", "y"]

combined = [*list1, *list2, *list3]
print(combined)
