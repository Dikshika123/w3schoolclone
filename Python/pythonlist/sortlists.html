Python Sort Lists

Sort List Alphanumerically

List objects have a sort() method that will sort the list alphanumerically, ascending, by default:

thislist = ["orange", "mango", "kiwi", "pineapple", "banana"]
thislist.sort()
print(thislist)

Sort Descending

To sort descending, use the keyword argument reverse = True:

thislist = ["orange", "mango", "kiwi", "pineapple", "banana"]
thislist.sort(reverse = True)
print(thislist)

Sort Numbers

You can sort numbers:

thislist = [100, 50, 65, 82, 23]
thislist.sort()
print(thislist)

Sort Numbers Descending

thislist = [100, 50, 65, 82, 23]
thislist.sort(reverse = True)
print(thislist)

Customize Sort Function

You can also customize your own function by using the keyword argument key = function.

The function will return a number that will be used to sort the list (the lowest number first):

def myfunc(n):
    return abs(n - 50)

thislist = [100, 50, 65, 82, 23]
thislist.sort(key = myfunc)
print(thislist)

Case Insensitive Sort

By default the sort() method is case sensitive, resulting in all capital letters being sorted before lower case letters:

thislist = ["banana", "Orange", "Kiwi", "cherry"]
thislist.sort()
print(thislist)

Luckily we can use built-in functions as key functions when sorting the list.

So if you want a case-insensitive sort function, use str.lower as a key function:

thislist = ["banana", "Orange", "Kiwi", "cherry"]
thislist.sort(key = str.lower)
print(thislist)

Reverse Order

What if you want to reverse the order of a list, regardless of the alphabet?

The reverse() method reverses the current sorting order of the elements.

thislist = ["banana", "Orange", "Kiwi", "cherry"]
thislist.reverse()
print(thislist)

Sort by Length

You can sort by the length of the strings:

thislist = ["apple", "banana", "cherry", "kiwi", "mango"]
thislist.sort(key = len)
print(thislist)

Sort by Multiple Criteria

You can sort by multiple criteria:

def sort_criteria(item):
    return (len(item), item.lower())

thislist = ["apple", "Banana", "cherry", "kiwi", "Mango"]
thislist.sort(key = sort_criteria)
print(thislist)

Sort Tuples in List

You can sort lists containing tuples:

students = [("Alice", 85), ("Bob", 92), ("Charlie", 78)]
students.sort(key = lambda x: x[1], reverse = True)  # Sort by grade
print(students)

Using sorted() Function

The sorted() function returns a sorted list without modifying the original:

thislist = ["orange", "mango", "kiwi", "pineapple", "banana"]
sorted_list = sorted(thislist)
print(sorted_list)
print(thislist)  # Original list unchanged
