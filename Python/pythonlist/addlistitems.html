Python Add List Items

Append Items

To add an item to the end of the list, use the append() method:

thislist = ["apple", "banana", "cherry"]
thislist.append("orange")
print(thislist)

Insert Items

To insert a list item at a specified index, use the insert() method.

The insert() method inserts an item at the specified index:

thislist = ["apple", "banana", "cherry"]
thislist.insert(1, "orange")
print(thislist)

Extend List

To append elements from another list to the current list, use the extend() method.

thislist = ["apple", "banana", "cherry"]
tropical = ["mango", "pineapple", "papaya"]
thislist.extend(tropical)
print(thislist)

Add Any Iterable

The extend() method does not have to append lists, you can add any iterable object (tuples, sets, dictionaries etc.):

thislist = ["apple", "banana", "cherry"]
thistuple = ("kiwi", "orange")
thislist.extend(thistuple)
print(thislist)

Add Multiple Items

You can add multiple items at once using the + operator:

thislist = ["apple", "banana", "cherry"]
thislist = thislist + ["orange", "grape", "kiwi"]
print(thislist)

Add Items at Specific Position

You can insert multiple items at a specific position:

thislist = ["apple", "banana", "cherry"]
thislist[1:1] = ["orange", "grape"]
print(thislist)

Add Items Using List Concatenation

You can use the += operator to add items:

thislist = ["apple", "banana", "cherry"]
thislist += ["orange", "grape"]
print(thislist)

Add Items from User Input

You can add items from user input:

fruits = ["apple", "banana"]

# Add single item
new_fruit = "cherry"
fruits.append(new_fruit)

# Add multiple items
more_fruits = ["orange", "grape"]
fruits.extend(more_fruits)

print(fruits)

Add Items Conditionally

You can add items based on conditions:

numbers = [1, 2, 3]

# Add even numbers only
for i in range(4, 10):
    if i % 2 == 0:
        numbers.append(i)

print(numbers)
