Python Modules

A module is a file containing Python definitions and statements. It allows you to organize your code into reusable components.

What is a Module?

A module is simply a Python file with a .py extension that contains Python code. You can import and use functions, classes, and variables from modules.

Creating a Module

# Save as mymodule.py
def greeting(name):
    return f"Hello, {name}!"

def add_numbers(a, b):
    return a + b

person = {
    "name": "John",
    "age": 30,
    "country": "Norway"
}

Importing a Module

# Import the entire module
import mymodule

print(mymodule.greeting("Alice"))
print(mymodule.add_numbers(5, 3))
print(mymodule.person["name"])

Import Specific Functions

# Import specific functions
from mymodule import greeting, add_numbers

print(greeting("Bob"))
print(add_numbers(10, 20))

Import with Alias

# Import with an alias
import mymodule as mm

print(mm.greeting("Charlie"))
print(mm.add_numbers(15, 25))

Import All Functions

# Import all functions (not recommended)
from mymodule import *

print(greeting("David"))
print(add_numbers(30, 40))

Built-in Modules

# Import built-in modules
import datetime
import math
import random

# Use datetime module
x = datetime.datetime.now()
print(x)

# Use math module
print(math.sqrt(16))  # 4.0
print(math.pi)        # 3.141592653589793

# Use random module
print(random.randint(1, 10))  # Random number between 1 and 10

Module Search Path

import sys
print(sys.path)  # Shows where Python looks for modules

Creating a Package

A package is a directory containing multiple modules. To create a package:

  1. Create a directory
  2. Add an __init__.py file (can be empty)
  3. Add your module files

Package Structure

mypackage/
    __init__.py
    module1.py
    module2.py
    subpackage/
        __init__.py
        module3.py

Importing from Packages

# Import from package
from mypackage import module1
from mypackage.module2 import some_function

# Import from subpackage
from mypackage.subpackage import module3

__name__ Variable

# In mymodule.py
def greeting(name):
    return f"Hello, {name}!"

if __name__ == "__main__":
    print("This runs only when the module is executed directly")
    print("Not when imported")

Reloading a Module

import importlib
import mymodule

# Reload the module
importlib.reload(mymodule)
