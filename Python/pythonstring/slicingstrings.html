Python Slicing Strings

Slicing

You can return a range of characters by using the slice syntax.

Specify the start index and the end index, separated by a colon, to return a part of the string.

b = "Hello, World!"
print(b[2:5])

Slice From the Start

By leaving out the start index, the range will start at the first character:

b = "Hello, World!"
print(b[:5])

Slice To the End

By leaving out the end index, the range will go to the end:

b = "Hello, World!"
print(b[2:])

Negative Indexing

Use negative indexes to start the slice from the end of the string:

b = "Hello, World!"
print(b[-5:-2])

Step in Slicing

You can specify a step value as the third parameter:

b = "Hello, World!"
print(b[2:5:2])  # Every 2nd character

Reverse String

You can reverse a string using slicing:

b = "Hello, World!"
print(b[::-1])  # Reverse the string

More Slicing Examples

text = "Python Programming"

# Get first 6 characters
print(text[:6])  # Output: Python

# Get last 11 characters
print(text[-11:])  # Output: Programming

# Get every 2nd character
print(text[::2])  # Output: Pto rgamn

# Get characters from index 7 to 11
print(text[7:11])  # Output: Prog
