Python Format Strings

String Format

As we learned in the Python Variables chapter, we cannot combine strings and numbers like this:

age = 36
txt = "My name is John, I am " + age
print(txt)

But we can combine strings and numbers by using the format() method!

The format() method takes the passed arguments, formats them, and places them in the string where the placeholders {} are:

age = 36
txt = "My name is John, and I am {}".format(age)
print(txt)

The format() method takes unlimited number of arguments, and are placed into the respective placeholders:

quantity = 3
itemno = 567
price = 49.95
myorder = "I want {} pieces of item {} for {} dollars.".format(quantity, itemno, price)
print(myorder)

You can use index numbers {0} to be sure the arguments are placed in the correct placeholders:

quantity = 3
itemno = 567
price = 49.95
myorder = "I want to pay {2} dollars for {0} pieces of item {1}.".format(quantity, itemno, price)
print(myorder)

Named Indexes

You can also use named indexes by entering a name inside the curly brackets {carname}, but then you must use names when you pass the parameter values txt.format(carname = "Ford"):

myorder = "I have a {carname}, it is a {model}.".format(carname = "Ford", model = "Mustang")
print(myorder)

F-String Formatting (Python 3.6+)

F-strings provide a more readable way to format strings:

name = "Alice"
age = 25
message = f"My name is {name} and I am {age} years old"
print(message)

F-String with Expressions

You can use expressions inside f-strings:

price = 19.99
quantity = 3
total = f"Total: ${price * quantity:.2f}"
print(total)

String Formatting with % Operator

The % operator can also be used for string formatting:

name = "John"
age = 36
txt = "My name is %s, and I am %d years old" % (name, age)
print(txt)

Formatting Numbers

You can format numbers with specific decimal places:

number = 3.14159

# Format to 2 decimal places
formatted = "{:.2f}".format(number)
print(formatted)

# Format with f-string
formatted_f = f"{number:.2f}"
print(formatted_f)
