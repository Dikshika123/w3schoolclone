Python Concatenate Strings

String Concatenation

To concatenate, or combine, two strings you can use the + operator.

a = "Hello"
b = "World"
c = a + b
print(c)

Add a space between them

To add a space between them, add a " ":

a = "Hello"
b = "World"
c = a + " " + b
print(c)

String Concatenation with Variables

You can also concatenate strings with variables:

first_name = "John"
last_name = "Doe"
full_name = first_name + " " + last_name
print(full_name)

Concatenate Multiple Strings

You can concatenate multiple strings at once:

greeting = "Hello"
name = "Alice"
message = greeting + ", " + name + "! How are you?"
print(message)

String Concatenation with Numbers

To concatenate a string with a number, you need to convert the number to a string first:

age = 25
text = "I am " + str(age) + " years old"
print(text)

Using the += Operator

You can also use the += operator to concatenate strings:

message = "Hello"
message += " World"
print(message)

Concatenate with join() Method

The join() method is an efficient way to concatenate multiple strings:

words = ["Hello", "World", "Python"]
sentence = " ".join(words)
print(sentence)

Concatenate with Different Separators

You can use different separators with the join() method:

words = ["apple", "banana", "cherry"]

# Join with comma and space
result1 = ", ".join(words)
print(result1)

# Join with hyphen
result2 = "-".join(words)
print(result2)

# Join with newline
result3 = "\n".join(words)
print(result3)
