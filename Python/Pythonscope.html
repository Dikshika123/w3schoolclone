Python Scope

Scope refers to the region of a program where a variable is accessible.

Types of Scope

  • Local Scope: Variables defined inside a function
  • Enclosing Scope: Variables in the enclosing function (nested functions)
  • Global Scope: Variables defined at the top level of a module
  • Built-in Scope: Names in the built-in module

Local Scope

def my_function():
    x = 300  # Local variable
    print(x)

my_function()
# print(x)  # This would cause an error - x is not defined globally

Global Scope

x = 300  # Global variable

def my_function():
    print(x)  # Can access global variable

my_function()
print(x)  # Can access global variable

Global Keyword

x = 300

def my_function():
    global x
    x = 200  # This changes the global variable
    print(x)

my_function()
print(x)  # Now x is 200

Nonlocal Keyword

def outer_function():
    x = 300  # Enclosing scope
    
    def inner_function():
        nonlocal x
        x = 200  # This changes the enclosing scope variable
        print(x)
    
    inner_function()
    print(x)  # Now x is 200

outer_function()

Scope Example

# Global scope
global_var = "I'm global"

def outer_function():
    # Enclosing scope
    enclosing_var = "I'm enclosing"
    
    def inner_function():
        # Local scope
        local_var = "I'm local"
        print(global_var)    # Can access global
        print(enclosing_var) # Can access enclosing
        print(local_var)     # Can access local
    
    inner_function()

outer_function()

Variable Shadowing

x = 10  # Global x

def my_function():
    x = 20  # Local x shadows global x
    print(f"Local x: {x}")

my_function()
print(f"Global x: {x}")  # Global x is unchanged

Built-in Scope

# Built-in functions are in built-in scope
print(len("Hello"))  # len is a built-in function
print(max(1, 2, 3))  # max is a built-in function

# You can override built-in names (not recommended)
len = 5
# print(len("Hello"))  # This would cause an error now

Scope Resolution Order (LEGB)

Python follows the LEGB rule when looking for variables:

  1. Local scope
  2. Enclosing scope
  3. Global scope
  4. Built-in scope
