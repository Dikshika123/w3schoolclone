Python Classes and Objects

Classes are like blueprints for creating objects. Objects are instances of classes.

Creating a Class

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def greet(self):
        return f"Hello, I'm {self.name} and I'm {self.age} years old"

Creating an Object

# Create an object
person1 = Person("Alice", 25)
person2 = Person("Bob", 30)

# Access object properties
print(person1.name)  # Alice
print(person1.age)   # 25

# Call object methods
print(person1.greet())  # Hello, I'm Alice and I'm 25 years old

The __init__ Method

The __init__ method is called automatically when an object is created. It's used to initialize object properties.

Self Parameter

The self parameter refers to the current instance of the class. It must be the first parameter of any method in the class.

Modify Object Properties

person1.age = 26
print(person1.age)  # 26

Delete Object Properties

del person1.age
# print(person1.age)  # This will cause an error

Delete Objects

del person1
# person1 is no longer accessible
