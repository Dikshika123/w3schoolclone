Python PIP

PIP is a package manager for Python packages. It allows you to install and manage additional libraries and dependencies.

What is PIP?

PIP (Pip Installs Packages) is the standard package manager for Python. It comes pre-installed with Python 3.4+ and allows you to install packages from the Python Package Index (PyPI).

Check PIP Version

pip --version

Install a Package

# Install a package
pip install package_name

# Example: Install requests
pip install requests

Install Specific Version

# Install specific version
pip install package_name==version

# Example: Install requests version 2.25.1
pip install requests==2.25.1

Install from Requirements File

# Install from requirements.txt
pip install -r requirements.txt

List Installed Packages

# List all installed packages
pip list

# List outdated packages
pip list --outdated

Show Package Information

# Show package information
pip show package_name

# Example: Show requests information
pip show requests

Uninstall a Package

# Uninstall a package
pip uninstall package_name

# Example: Uninstall requests
pip uninstall requests

Upgrade a Package

# Upgrade a package
pip install --upgrade package_name

# Example: Upgrade requests
pip install --upgrade requests

Upgrade PIP

# Upgrade PIP itself
pip install --upgrade pip

Search for Packages

# Search for packages
pip search package_name

# Note: Search functionality was removed in newer versions
# Use PyPI website instead: https://pypi.org/

Freeze Installed Packages

# Generate requirements.txt
pip freeze > requirements.txt

# This creates a file with all installed packages and their versions

Install from Local Directory

# Install from local directory
pip install /path/to/package

# Install in development mode
pip install -e /path/to/package

Virtual Environments

# Create virtual environment
python -m venv myenv

# Activate virtual environment (Windows)
myenv\Scripts\activate

# Activate virtual environment (Linux/Mac)
source myenv/bin/activate

# Deactivate virtual environment
deactivate

Common Packages

  • requests: HTTP library for making web requests
  • numpy: Numerical computing library
  • pandas: Data manipulation and analysis
  • matplotlib: Plotting library
  • django: Web framework
  • flask: Lightweight web framework
  • beautifulsoup4: Web scraping library
  • selenium: Web browser automation

PIP Configuration

# Create pip.conf file
# Windows: %APPDATA%\pip\pip.ini
# Linux/Mac: ~/.pip/pip.conf

[global]
index-url = https://pypi.org/simple/
trusted-host = pypi.org
timeout = 60
