Python Iterators

An iterator is an object that contains a countable number of values and can be iterated upon.

What is an Iterator?

An iterator is an object that implements the iterator protocol, which consists of the methods __iter__() and __next__().

Iterator vs Iterable

  • Iterable: An object that can return an iterator (lists, tuples, strings, etc.)
  • Iterator: An object that implements the iterator protocol

Creating an Iterator

class MyNumbers:
    def __init__(self):
        self.a = 1
        self.b = 1
    
    def __iter__(self):
        return self
    
    def __next__(self):
        if self.a <= 20:
            x = self.a
            self.a += 1
            return x
        else:
            raise StopIteration

myclass = MyNumbers()
myiter = iter(myclass)

for x in myiter:
    print(x)

Using Built-in Iterators

# List iterator
mylist = [1, 2, 3, 4, 5]
myiter = iter(mylist)

print(next(myiter))  # 1
print(next(myiter))  # 2
print(next(myiter))  # 3

String Iterator

mystr = "Hello"
myiter = iter(mystr)

for x in myiter:
    print(x)  # H, e, l, l, o

Tuple Iterator

mytuple = ("apple", "banana", "cherry")
myiter = iter(mytuple)

for x in myiter:
    print(x)

Creating a Custom Iterator

class Counter:
    def __init__(self, low, high):
        self.current = low
        self.high = high
    
    def __iter__(self):
        return self
    
    def __next__(self):
        if self.current < self.high:
            num = self.current
            self.current += 1
            return num
        raise StopIteration

for x in Counter(1, 6):
    print(x)  # 1, 2, 3, 4, 5

StopIteration Exception

The StopIteration exception is raised when the iterator has no more values to return. This is handled automatically by the for loop.
